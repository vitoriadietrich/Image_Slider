# Image Slider in HTML CSS & JavaScript | Card Slider

Um projeto simples de **image/card slider** usando HTML, CSS e JavaScript.

## ✨ Sobre

Este projeto demonstra como criar um **slider de imagens ou cards interativo**, ideal para galerias, produtos ou portfólios.

## 🛠️ Tecnologias

- HTML
- CSS
- JavaScript

## 🎯 Funcionalidades

- Slider interativo
- Navegação entre itens
- Design responsivo e moderno

## 🚀 Como usar

1. Baixe ou clone o repositório  
2. Abra o arquivo `index.html` no navegador
